# 🌱 Smart Irrigation System

## Description
This project is an IoT-based smart irrigation system using Raspberry Pi.

## Features
- Soil moisture sensing
- Temperature & humidity monitoring
- Rain detection
- Motor control (automatic/manual)

## Technologies Used
- Python
- Raspberry Pi
- Sensors (DHT11, Soil Moisture, Rain Sensor)

## How to Run
1. Connect all sensors to Raspberry Pi
2. Run the script:
   python3 irrigation.py
